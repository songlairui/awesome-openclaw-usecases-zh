# 定制晨报

你醒来后花前 30 分钟赶进度——刷新闻、查看日历、审查待办事项列表，试图弄清楚今天什么重要。如果所有这些都已经完成并作为短信等着你呢？

这个工作流让 OpenClaw 每天在预定时间向你发送完全定制的晨报，涵盖新闻、任务、想法和主动建议。

## 功能说明

- 每天在同一时间（例如早上 8:00）向 Telegram、Discord 或 iMessage 发送结构化的晨报
- 通过浏览网络研究与你兴趣相关的夜间新闻
- 审查你的待办事项列表并浮现当天的任务
- 在你睡觉时生成创意输出（完整脚本、电子邮件草稿、商业提案——不仅仅是想法）
- 推荐 AI 可以自主完成以帮助你的任务

## 痛点

你在最有生产力的早晨时间里只是在适应。与此同时，你的 AI 智能体整夜闲置。晨报将闲置的夜间时间转化为富有成效的准备时间——你醒来时工作已经完成。

## 所需技能

- Telegram、Discord 或 iMessage 集成
- Todoist / Apple Reminders / Asana 集成（你用于任务的任何工具）
- [x-research-v2](https://clawhub.ai) 用于社交媒体趋势研究（可选）

## 设置方法

1. 将 OpenClaw 连接到你的消息平台和任务管理器。

2. 提示 OpenClaw：
```text
我想设置定期晨报。每天早上 8:00，
通过 Telegram 向我发送报告。

我希望此报告包括：
1. 与我的兴趣相关的新闻故事（AI、创业公司、科技）
2. 我今天可以创建的内容想法
3. 我今天需要完成的任务（从我的待办事项列表中提取）
4. 你今天可以为我完成的任务的建议

对于内容想法，写完整的草稿脚本/大纲——不仅仅是标题。
```

3. OpenClaw 将自动安排此操作。通过检查第二天早上的消息来验证它是否正常工作。

4. 随着时间的推移进行定制——只需给你的机器人发短信：
```text
在我的晨报中添加天气预报。
停止包括一般新闻，只关注 AI。
每天早上包括一句励志名言。
```

5. 如果你想不出要包括什么，你不必——只需说：
```text
我希望此报告包括与我相关的内容。
想想在这份报告中放什么最有帮助。
```

## 关键见解

- AI 推荐的任务部分是最强大的部分——它让智能体主动思考如何帮助你，而不是等待指令。
- 你可以通过发短信来定制简报。说"在我的晨报中添加股票价格"，它就会更新。
- 完整草稿（而不仅仅是想法）是节省时间的关键。醒来时看到脚本，而不是建议。
- 无论你从事什么行业——包含任务、新闻和主动建议的晨报都是普遍有用的。
