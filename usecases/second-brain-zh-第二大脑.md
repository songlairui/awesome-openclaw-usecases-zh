# 第二大脑

你想出想法、找到有趣的链接、听说要读的书——但你从来没有一个好的系统来捕捉它们。Notion 变得复杂，Apple Notes 变成了 10,000 个未读条目的墓地。你需要像给朋友发短信一样简单的东西。

这个工作流将 OpenClaw 变成一个通过短信交互的记忆捕获系统，由你可以随时浏览的自定义可搜索 UI 支持。

## 功能说明

- 通过 Telegram、iMessage 或 Discord 向你的 OpenClaw 发送任何内容——"提醒我阅读一本关于本地 LLM 的书"——它会立即记住
- OpenClaw 的内置记忆系统永久存储你告诉它的所有内容
- 自定义 Next.js 仪表板让你搜索每个记忆、对话和笔记
- 跨所有记忆、文档和任务的全局搜索（Cmd+K）
- 没有文件夹、没有标签、没有复杂的组织——只有文本和搜索

## 痛点

每个笔记应用最终都会变成一件苦差事。你停止使用它，因为组织的摩擦高于遗忘的摩擦。关键见解是：**捕获应该像发短信一样简单，检索应该像搜索一样简单**。

## 所需技能

- Telegram、iMessage 或 Discord 集成（用于基于文本的捕获）
- Next.js（OpenClaw 为你构建——无需编码）

## 设置方法

1. 确保你的 OpenClaw 连接到你首选的消息平台（Telegram、Discord 等）。

2. 立即开始使用——只需给你的机器人发短信，告诉它你想记住的任何内容：
```text
嘿，提醒我阅读"设计数据密集型应用程序"
保存此链接：https://example.com/interesting-article
记住：John 推荐了第 5 街的餐厅
```

3. 通过提示 OpenClaw 构建可搜索的 UI：
```text
我想构建一个第二大脑系统，我可以在其中查看我们所有的笔记、
对话和记忆。请用 Next.js 构建它。

包括：
- 所有记忆和对话的可搜索列表
- 跨所有内容的全局搜索（Cmd+K）
- 按日期和类型过滤的能力
- 干净、简约的 UI
```

4. OpenClaw 将为你构建和部署整个 Next.js 应用。导航到它提供的 URL，你就有了第二大脑仪表板。

5. 从现在开始，每当你想到什么——在路上、在会议中、睡前——只需给你的机器人发短信。每当你需要找到什么时，回到仪表板。
