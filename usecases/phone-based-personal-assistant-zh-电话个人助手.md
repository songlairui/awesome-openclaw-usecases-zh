# 基于电话的个人助手

## 痛点

你想从任何电话访问你的 AI 智能体，而无需智能手机应用或互联网浏览器。你需要在驾驶、步行或双手被占用时进行免提语音协助。

## 功能说明

ClawdTalk 使 OpenClaw 能够接听和拨打电话，将任何电话变成你的 AI 助手的网关。你可以：
- 拨打电话号码通过语音与你的 AI 智能体交谈
- 通过语音获取日历提醒、Jira 更新和网络搜索结果
- 与 Telnyx 集成以实现可靠的电话连接

SMS 支持即将推出。

## 提示词

```
你可以通过电话访问。当我打电话时，问候我并询问我如何帮助。

对于日历查询："我今天的日历上有什么？"
对于 Jira 更新："显示我的未解决工单"
对于网络搜索："搜索 AI 智能体的最新新闻"
```

## 所需技能

- [ClawdTalk](https://github.com/team-telnyx/clawdtalk-client)
- 日历技能（Google Calendar 或 Outlook）
- Jira 技能
- 网络搜索技能
