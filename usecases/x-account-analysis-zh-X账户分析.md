# X 账户分析

有许多网站旨在为你提供 X 账户的定性分析。虽然 X 已经为你提供了 **分析** 部分，但它更侧重于显示你的表现数字。

但定性分析侧重于你的帖子质量，而不是表现统计数据。你可以从这种类型的分析中获得一些见解：
- 是什么模式让我的帖子走红？
- 我谈论的哪些话题让我获得最多参与？
- 为什么我有些帖子获得 1000+ 点赞，但有时帖子 <5 点赞？我做错了什么？

有许多网站和应用旨在为你提供 X 分析，但它们专注于统计数据。可能有 1-2 个网站让你与 AI 交谈以了解你的表现。

但现在你可以使用 OpenClaw 为你进行此分析，而无需为这些网站的订阅支付 $10-$50。

## 所需技能
Bird 技能。`clawhub install bird`（它预先捆绑）

## 设置方法
流程如下：
1. 确保 Bird 技能正常工作。
2. 为了安全和隔离，你最好为你的 ClawdBot 创建一个新账户。
3. 使用你的 X 账户进行身份验证。在 Chrome/Brave 中登录 x.com，并提供正确的 cookie 信息（`auth-token`、`ct0`），以便 OpenClaw 可以访问你的账户。
4. 要求 OpenClaw 查看你的真实账户，获取最后 N 条推文，并向它提出你喜欢的任何问题。或者，你可以要求它为你编写特定的脚本。
