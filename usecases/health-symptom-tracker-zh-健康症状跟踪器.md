# 健康与症状跟踪器

识别食物敏感性需要长期一致的记录，这很繁琐。你需要提醒来记录并分析以发现模式。

这个工作流自动跟踪食物和症状：

• 在专用的 Telegram 主题中发送你的食物和症状消息，OpenClaw 会记录所有内容并附带时间戳
• 每天 3 次提醒（早上、中午、晚上）提示你记录餐食
• 随着时间的推移，分析模式以识别潜在触发因素

## 所需技能

- Cron 作业用于提醒
- Telegram 主题用于记录
- 文件存储（markdown 日志文件）

## 设置方法

1. 创建一个名为"health-tracker"（或类似）的 Telegram 主题。
2. 创建日志文件：`~/clawd/memory/health-log.md`
3. 提示 OpenClaw：
```text
当我在"health-tracker"主题中发消息时：
1. 解析消息中的食物项目和症状
2. 记录到 ~/clawd/memory/health-log.md 并附带时间戳
3. 确认记录了什么

设置 3 个每日提醒：
- 早上 8 点："🍳 记录你的早餐"
- 下午 1 点："🥗 记录你的午餐"
- 晚上 7 点："🍽️ 记录你的晚餐和任何症状"

每周日，分析过去一周的日志并识别模式：
- 哪些食物与症状相关？
- 是否有一天中的时间模式？
- 有明显的触发因素吗？

将分析发布到 health-tracker 主题。
```

4. 可选：为 OpenClaw 添加一个记忆文件来跟踪已知触发因素，并在模式出现时更新它。
