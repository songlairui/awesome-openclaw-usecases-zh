# 多智能体内容工厂

你是一个内容创作者，需要在多个平台上处理研究、写作和设计工作。每个步骤——寻找热门话题、写脚本、生成缩略图——都要花费大量时间。如果有一个专业智能体团队能在夜间处理所有这些工作呢？

这个工作流在 Discord 中建立了一个多智能体内容工厂，不同的智能体在专门的频道中处理研究、写作和视觉资源。

## 功能说明

- **研究智能体** 每天早上扫描热门故事、竞争对手内容和社交媒体，寻找最佳内容机会
- **写作智能体** 选取最好的想法，撰写完整的脚本、推文串或新闻稿草稿
- **缩略图智能体** 为内容生成 AI 缩略图或封面图片
- 每个智能体在自己的 Discord 频道中工作，保持一切井然有序且可审查
- 按计划自动运行（例如每天早上 8 点），让你醒来时就有完成的内容

## 痛点

内容创作有三个阶段——研究、写作和设计——大多数创作者都在手动完成这三个阶段。即使有 AI 写作工具，你仍然需要逐个提示它们。这个系统将智能体串联成一个流水线，一个智能体的输出成为下一个的输入，完全无需人工干预。

## 所需技能

- Discord 集成，支持多个频道
- `sessions_spawn` / `sessions_send` 用于多智能体编排
- [x-research-v2](https://clawhub.ai) 或类似工具用于社交媒体研究
- 本地图像生成（例如 Nano Banana）或图像生成 API
- [knowledge-base](https://clawhub.ai) 技能（可选，用于 RAG 驱动的研究）

## 设置方法

1. 建立一个 Discord 服务器（或让 OpenClaw 帮你做——只需说"给我们建立一个 Discord"）。

2. 为每个智能体创建频道：
   - `#research` — 热门话题和内容机会
   - `#scripts` — 撰写的草稿和大纲
   - `#thumbnails` — 生成的图片和封面艺术

3. 提示 OpenClaw：
```text
我想让你在 Discord 中为我建立一个内容工厂。
为不同的智能体设置频道：

1. 研究智能体（#research）：每天早上 8 点，研究热门故事、
   竞争对手内容，以及我所在领域在社交媒体上表现良好的内容。
   发布前 5 个内容机会及来源。

2. 写作智能体（#scripts）：从研究智能体中选取最好的想法，
   撰写完整的脚本/推文串/新闻稿草稿。发布到 #scripts。

3. 缩略图智能体（#thumbnails）：为内容生成 AI 缩略图或封面图片。
   发布到 #thumbnails。

将他们的工作组织在不同的频道中。
每天早上自动运行这个流水线。
```

4. 根据你的平台定制：
```text
我专注于 X/Twitter 推文串，而不是 YouTube。将写作智能体改为
生成推文串而不是视频脚本。
```

## 关键见解

- 力量在于**串联智能体** — 研究为写作提供素材，写作为缩略图提供素材。你不需要单独提示每个步骤。
- Discord 频道让你可以分别审查每个智能体的工作，并给出反馈，比如"脚本太长"或"更多关注 AI 新闻"。
- 你可以将此适配到任何内容格式：推文、新闻稿、LinkedIn 帖子、播客大纲、博客文章。
- 运行本地模型进行图像生成（比如 Mac Studio 上的 Nano Banana）可以降低成本并提供更多控制。
