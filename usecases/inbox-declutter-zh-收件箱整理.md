# 收件箱整理

新闻稿会像其他任何东西一样占据收件箱。它们经常堆积而根本不被打开。

## 所需技能
[Gmail OAuth 设置](https://clawhub.ai/kai-jar/gmail-oauth)。

## 设置方法
1. [可选] 专门为 OpenClaw 创建一个新的 gmail。
2. [可选] 从你的主邮箱取消订阅所有新闻稿，并使用 OpenClaw 邮箱订阅它们。
3. 安装技能并确保它有效。
4. 指示 OpenClaw：
```txt
我想让你每天晚上 8 点运行一个 cron 作业，阅读过去 24 小时的所有新闻稿邮件，并给我一个最重要内容的摘要以及阅读更多的链接。然后询问我对你选择的内容的反馈，并根据我的偏好更新你的记忆，以便在未来的作业中获得更好的摘要。
```
